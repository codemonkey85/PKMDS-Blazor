PKMDS: Pokémon Save Editor for Web

Your browser isn't recent enough to run PKMDS. We need WebAssembly with the SIMD extension — used to accelerate save-file parsing, encryption, and legality checks.

Any of these (or newer) will work:

  • Chrome / Edge / Brave / Opera 91+ (May 2021)
  • Firefox 89+ (June 2021)
  • Safari 16.4+ on macOS, iPadOS, or iOS (March 2023)

If you're on an older iPhone or iPad that can't update to iOS 16.4 or later, unfortunately Apple no longer ships browser updates for that device and PKMDS can't run there.

Want the gritty details? Check caniuse.com/wasm-simd.
